Folder setup

This is the easiest way to get routes like solixo.dev/sky-dash on GitHub Pages or any static host.

Project structure
index.html sky-dash/ index.html future-game/ index.html another-game/ index.html
Root index.html This hub page lives at the root of the site, so it becomes solixo.dev.
Each game gets its own folder Put your game HTML inside a folder with its own index.html. That creates a clean route automatically.
Example route If your game lives in sky-dash/index.html, the route becomes solixo.dev/sky-dash.

How to add more later

You do not need to regenerate the whole site every time. Just keep the hub file, add a new folder, then copy one card and update the link.

Step 1 Create a new folder for the game, like boss-runner.
Step 2 Put that game inside boss-runner/index.html.
Step 3 Duplicate one of the game cards in the hub and change the link to ./boss-runner/.
Example card link
<a class="gameCard cardBlue" href="./boss-runner/"> ... </a>
